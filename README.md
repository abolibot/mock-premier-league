## Getting Started

### Clone Repository

```git
    git clone https://github.com/abolibot/mock-premier-league.git
```

### Install Dependencies

```bash
    pnpm install
```

### Start Development Server

```bash
    pnpm run watch
```

### Build

```bash
    pnpm run build
```
